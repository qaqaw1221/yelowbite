Как выложить сайт бесплатно (GitHub Pages)

Подходит для простых сайтов на HTML/CSS/JS. Вы загружаете файлы — и получаете постоянную ссылку.

Реклама
Место под баннер
Замените на код рекламной сети.
К статьям

Что нужно заранее

  • Файлы сайта: минимум index.html (и при желании styles.css).
  • Аккаунт GitHub: бесплатный.
  • Название репозитория: например yelowbite или my-site.

Пошагово

  1. Создайте репозиторий на GitHub (New repository).
  2. Загрузите файлы сайта (Upload files): index.html, styles.css.
  3. Откройте Settings → Pages:
    • Branch: main
    • Folder: /(root)
  4. Нажмите Save и подождите 1–2 минуты.
  5. Готово — GitHub даст ссылку вида https://ваш-ник.github.io/имя-репозитория/

Частые проблемы

Сайт пустой / 404: проверь, что файл называется именно index.html и лежит в корне репозитория.
Не обновляется дизайн: нажми Ctrl+F5 (браузер иногда кэширует styles.css).
